# 🚗 小T摸鱼大冒险

> 一个关于"小T 开车从腾讯北京总部偷偷溜去亚洲金融大厦找小L"的网页互动小游戏。

## 🎮 在线试玩

**[👉 点这里玩](https://你的用户名.github.io/你的仓库名/)**

## ✨ 玩法

- 🧭 看右上角**智能罗盘**指针方向
- 📍 在每个路口选择正确的方向（基于真实北京地图：腾讯总部 → 五环 → 北五环 → S50 → 清河 → 亚洲金融大厦）
- 🏢 沿途打卡**8 个北京真实地标**：腾讯总部 / 小米科技园 / 百度大厦 / 北体大 / 清华 / 圆明园 / 冰丝带 / 鸟巢
- 🌧️ 还会经过雨天打滑、堵车、修路、临时检查站等路障
- 😢 走错路 → 小L 哭哭表情 + 文案"迷路了，找不到我喽"
- 🎉 通关 → 小T + 小L 在亚洲金融大厦前合影

## 🎨 技术栈

- 纯静态网页，无需任何后端
- HTML / CSS / 原生 JavaScript
- 6 层视差滚动 + SVG 动画
